Tutorial de navegación de video

Navegar

Para seleccionar un video, pulsa el botón del centro.

Para navegar, utiliza las flechas.

Para buscar un video, utiliza el botón de la lupa.

Buscar

Para buscar un video, puedes pulsar el botón de voz y empezar a hablar. También puedes escribir en el teclado.

Control de video

Para parar el video, presiona el botón de play/pause.

Para subir el volumen, presiona el botón de +. Para bajar el volumen, presiona el botón de -.

Para realizar una videollamada, presiona el botón de videollamada y selecciona un contacto. Para colgar la llamada, presiona el mismo botón.

Gestos

Para ocultar los botones del controlador de video, haz un círculo en la pantalla.

Para iniciar el movimiento, pulsa dos veces. Los movimientos disponibles son:

  • Inclinar el móvil hacia delante: sube el volumen.
  • Inclinar el móvil hacia atrás: baja el volumen.
  • Inclinar el móvil hacia la izquierda: retrocede 10 segundos.
  • Inclinar el móvil hacia la derecha: avanza 10 segundos.
